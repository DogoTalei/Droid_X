# Droid_X
Potente herramienta CLI de seguridad para Windows, escrita en Python. Realiza un análisis profundo de dispositivos Android vía ADB, escaneando apps en busca de permisos peligrosos y vectores de riesgo. Se integra con la API de VirusTotal para verificar reputación y permite la desinstalación selectiva. Distribuido como un ejecutable portable.
